This page is where the Challenge Designer sets up the Challenge.
  1. Edit the Challenge Settings. Next determine the number and length of the Rounds. Then edit the Score Settings.
  2. Create the Groups, there must be at least one Group. The the Teams, finally the individual players.
  3. Checkout the PageInfo items to determine which pages are displayed in the Challenge. Update any of the Help Topics and Notice Templates.
  4. (Optionally) Edit the Scheduler information. The default settings should be good.
  5. Edit the Game settings, enabling or disabling the games and setting their specific settings.
